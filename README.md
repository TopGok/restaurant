required php, nodejs, npm, composer.
step : run terminal or set config
1. npm i
2. composer i
3. set environment
    - create database
    - set .env to connect database
    - set .env CACHE_DRIVER=database
4. php artisan migrate
5. php artisan key:generate
6. php artisan serve
